# Dart 예외처리 실습 프로젝트

컬렉션, 함수, 파일 입출력, 객체, 상속에 대한 개념이 포인트인 과제입니다.

---

## 1. 성적 분석 프로그램

### ✔ 설명  
사용자로부터 학생 이름을 입력 받아 해당 학생의 이름과 점수가 출력이 되고,
result.txt 파일에 저장된다.

### ✔ 필수 기능
- 컬렉션을 활용하여 데이터 처리하기
- 파일 입출력을 통해 데이터를 불러오고 저장하기
- 객체 지향을 활용한 전체 구조 생각하기


## 2. 우수생을 출력하는 성적 분석 프로그램

### ✔ 설명  
사용자로부터 메뉴 입력을 받아 알맞는 분기 처리를 한다.

### ✔ 필수 기능
- 사용자가 1을 입력하면, 우수생(가장 높은 평균 점수를 가진 학생)의 정보를 출력합니다.
- 사용자가 2를 입력하면, 전체 학생들의 평균 점수를 계산하여 출력합니다.
